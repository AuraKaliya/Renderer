# 0.0.11 评审遗留项
## 1. 背景

`0.0.11` 主要完成正交模式下的缩放与聚焦语义收口：

- 正交模式默认主缩放改为 `Lens`
- 正交模式 `Focus Bounds` 改为拟合 `orthographicHeight`
- `Focus Point` 在正交模式下保持当前 `orthographicHeight`

当前版本结束后，最自然的后续评审主题已经明确为：

- `Perspective + Lens`
- 投影模式与缩放模式的完整 UI 策略

## 2. 发现时间

- 2026-03-12

## 3. 所属版本

- 作为 `0.0.11` 结束后的后续版本评审输入

## 4. 是否阻塞当前步骤

- 否

说明：

- 当前正交模式默认行为已经具备最小可用闭环
- `Perspective + Lens` 不阻塞本版本收口

## 5. 风险说明

- 若在当前阶段直接引入 `Perspective + Lens`，会同时影响：
  - 透视模式滚轮缩放习惯
  - `Focus All / Focus Point / Focus Sphere` 的现有 framing 认知
  - 控制面板的交互边界
- 若没有先明确 UI 策略，后续投影模式和缩放模式组合会继续堆叠在现有相机面板上

## 6. 建议处理版本

- 建议下一版本优先评审 `Perspective + Lens` 的行为边界
- UI 是否暴露 `Zoom Mode` 的主动切换，建议再单独评审

## 7. 建议方案

### 7.1 Perspective + Lens

建议优先评审：

1. `Perspective + Lens` 是否以修改 `verticalFovDegrees` 为唯一语义
2. 与当前 `Perspective + Dolly` 是否允许自由切换
3. `Focus Bounds` 在 `Perspective + Lens` 下是否仍以 `distance` 为主，还是改为 `FOV` 拟合

### 7.2 UI 方案

建议评审：

1. `Projection Mode` 是否继续保留直接切换
2. `Zoom Mode` 是否在 UI 中显式开放切换
3. `Distance / FOV / Ortho Height` 的启用禁用规则是否需要进一步明确

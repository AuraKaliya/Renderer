# 0.0.8 开发记录

## 1. 版本目标

围绕当前轨道相机的缩放与裁剪问题，补齐透视模式下 Dolly 缩放过程中的基础 `near/far` 联动，并建立后续扩展正交模式与多种缩放模式所需的状态接口基础。

## 2. 实际完成内容

### 2.1 拆分相机裁剪计算逻辑

- 新增 `camera_clip_utils`
- 将 `near/far` 计算从 `camera_focus_utils` 中独立出来
- 当前职责边界已拆分为：
  - `camera_focus_utils`
  - `camera_clip_utils`

### 2.2 补齐透视模式下 Dolly 缩放的 near/far 联动

- 滚轮缩放后会联动更新基础 `near/far`
- 当前联动优先基于“当前可见对象”的稳定聚焦范围
- 当没有可见对象时，退化为基于当前 `orbitCenter` 更新裁剪面

### 2.3 修复 Focus All / Focus Sphere 重复触发不稳定

- 修复了重复点击 `Focus All` 或 `Focus Sphere` 时：
  - `distance` 抖动
  - `near/far` 抖动
- 当前聚焦行为改为基于稳定聚焦 bounds，而不是动画瞬时 `world AABB`
- 该问题已归档到：
  - `DebugDoc/0.0.8-问题.md`

### 2.4 建立 ProjectionMode / ZoomMode 状态接口

- `OrbitCameraController` 新增：
  - `ProjectionMode`
  - `ZoomMode`
- 新增：
  - `setProjectionMode(...) / projectionMode()`
  - `setZoomMode(...) / zoomMode()`
  - `setOrthographicHeight(...) / orthographicHeight()`
- 当前默认行为保持：
  - `Perspective`
  - `Dolly`

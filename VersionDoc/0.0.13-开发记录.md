# 0.0.13 开发记录

## 1. 版本概述

`0.0.13` 围绕 `Zoom Mode` 的完整 UI 策略，对测试界面的相机面板和窗口层 UI 编排职责做了第二轮收口。

本版本没有继续扩相机新能力，而是优先处理：

- 相机 UI 规则表达
- 相机面板内部职责边界
- `ViewerWindow` 的 UI 同步与编排收口

## 2. 本版本完成内容

### 2.1 相机面板规则收口

完成对 `CameraControlWidget` 的内部结构整理：

- 将相机面板分为：
  - `Mode`
  - `Parameters`
  - `Focus / State`
- 对以下参数增加角色表达：
  - `Distance`
  - `Vertical FOV`
  - `Ortho Height`

角色包括：

- `Primary`
- `Secondary`
- `Inactive`

当前规则已明确覆盖：

- `Perspective + Dolly`
- `Perspective + Lens`
- `Orthographic + Lens`

### 2.2 ViewerWindow UI 编排收口

完成对 `ViewerWindow` 的 UI 同步职责整理：

- 增加 `bindControlPanelSignals()`
- 增加：
  - `syncSceneObjectPanel()`
  - `syncLightingPanel()`
  - `syncCameraPanel()`
- `syncControlPanel()` 只保留为总调度入口

同时新增：

- `Viewport::cameraPanelState()`
- `ViewerControlPanel::CameraPanelState`

使相机面板回填不再依赖 `ViewerWindow` 手工逐项拼装多个参数。

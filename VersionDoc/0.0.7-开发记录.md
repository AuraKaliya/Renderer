# 0.0.7 开发记录

## 1. 版本目标

在已有 `worldBounds / sceneBounds / orbitCenter / Focus Point / Focus All` 最小接口的基础上，正式将聚焦能力接入测试界面，并补齐基础的距离拟合与裁剪参数更新逻辑，形成可验证的聚焦闭环。

## 2. 实际完成内容

### 2.1 聚焦计算工具与基础裁剪更新

- 新增 `camera_focus_utils`
- 基于 `Aabb` 提供：
  - bounds center 计算
  - `Focus All` 距离拟合
  - `Focus All / Focus Point` 的基础 `near/far` 计算
- `OrbitCameraController::focusOnPoint()` 与 `focusOnBounds()` 已统一复用聚焦 helper

### 2.2 Focus All / Focus Point 接入测试界面

- `CameraControlWidget` 新增：
  - `Near / Far` 只读显示
  - `Point X / Y / Z` 输入
  - `Focus Point` 按钮
- `ViewerControlPanel` 的 `Actions` 区域新增：
  - `Focus All`
- `ViewerWindow` 已完成：
  - `Focus All`
  - `Focus Point`
  与视口状态的信号连接

### 2.3 Focus Sphere 收口到统一聚焦链路

- `Focus Sphere` 不再单独保留独立的相机摆位逻辑
- 现在统一通过目标 `worldBounds` 走聚焦 helper
- `Focus Sphere` 可同步更新：
  - `orbitCenter`
  - `distance`
  - `near/far`

### 2.4 Focus All 可见性收口

- `Focus All` 已改为基于“当前可见对象”计算范围
- 隐藏对象不再参与 `Focus All` 聚焦
- 当前没有可见对象时，相机状态保持不变

## 6. 已知问题与限制

- 当前滚轮缩放仍未联动更新 `near/far`
- 当前相机仍只有透视模式，尚未支持正交模式切换
- `ProjectionMode / ZoomMode` 仍未独立建模

## 7. 本版本遗留项

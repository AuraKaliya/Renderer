# 0.0.11 开发记录
## 1. 版本目标

围绕正交模式下的缩放与聚焦行为，补齐与透视模式分离的 helper 分支和控制器行为，使正交模式具备更自然的默认导航语义，并为后续 `Lens Zoom` 与多组合缩放模式扩展打基础。

## 2. 实际完成内容

### 2.1 拆分正交模式 focus helper 分支

- `camera_focus_utils` 新增：
  - `ProjectionMode`
  - `orthographicHeight`
  相关输出
- 正交模式下的 `Focus Bounds` 改为拟合 `orthographicHeight`
- 正交模式下的 `Focus Point` 保持当前 `orthographicHeight`
- 保留旧接口兼容包装，避免当前透视路径回归

### 2.2 调整控制器缩放与聚焦行为

- `OrbitCameraController` 在 `Orthographic` 模式下默认主缩放改为 `Lens`
- 正交模式滚轮缩放改为调整 `orthographicHeight`
- `Focus All / Focus Bounds / Focus Point` 根据投影模式走不同 helper 分支
- 透视模式当前 `Dolly` 逻辑保持不变

### 2.3 补最小验证入口并整理后续输入

- `Camera` 面板新增只读 `Zoom Mode`
- 当前界面会根据投影模式启用/禁用对应输入项：
  - `Perspective`
    - 启用 `Distance / Vertical FOV`
    - 禁用 `Ortho Height`
  - `Orthographic`
    - 启用 `Ortho Height`
    - 禁用 `Distance / Vertical FOV`
- 新增后续评审输入文档：
  - `ReviewDoc/0.0.11-评审遗留项.md`

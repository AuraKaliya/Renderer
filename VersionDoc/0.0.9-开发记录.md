# 0.0.9 开发记录
## 1. 版本目标

围绕当前相机与聚焦系统的代码结构进行整理，显式拆分 `render bounds` 与 `focus bounds` 语义，抽离稳定聚焦范围 helper，并收紧 `Viewport` 内部职责，为后续正交模式真实投影路径接入提供更清晰的结构输入。

## 2. 实际完成内容

### 2.1 显式拆分 focus bounds 语义与 helper

- 新增 `camera_focus_bounds_utils`
- 将稳定聚焦范围计算从 `Viewport` 中抽离到独立 helper
- 显式区分：
  - `render bounds`
  - `focus bounds`
- 统一使用 `objectFocusBounds(...)`、`visibleFocusBounds()` 命名，避免继续混用语义

### 2.2 收紧 Viewport 内部职责

- `Viewport` 新增统一调度入口：
  - `currentObjectTransform(int index)`
  - `notifyCameraStateChanged()`
  - `applyFocusBounds(...)`
- 对象当前变换构造收口到单一入口，避免多处重复展开对象状态
- `Focus All / Focus Sphere / Focus Point` 与滚轮缩放裁剪更新统一复用 helper 和调度入口
- `Viewport` 当前主要承担：
  - 状态调度
  - helper 调用
  - 渲染更新触发

### 2.3 为后续正交模式接入准备评审输入

- 新增 `ReviewDoc/0.0.9-评审遗留项.md`
- 明确后续优先输入：
  1. 正交模式真实投影路径接入
  2. `orthographicHeight` 状态链打通
  3. `Lens Zoom` 行为边界与 UI 方案延后评审
